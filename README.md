# Mountain View Booking System

โปรเจกต์ระบบจองที่พัก Mountain View  
พัฒนาด้วยภาษา Python (Flask) และ MySQL

## Features
- สมัครสมาชิก / เข้าสู่ระบบ
- จองที่พัก
- ดูรายการจอง
- อัปโหลดสลิปการชำระเงิน
- ระบบผู้ดูแลจัดการการจอง

## Technologies
- Python Flask
- MySQL
- HTML / CSS
- Bootstrap

## Installation

1. Clone project
2. Import database จากไฟล์ database/mountain_view.sql
3. ติดตั้ง dependencies
